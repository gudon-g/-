## 使用方法：
点击GO按钮即可！自动运行完成后自动停止！

## 使用技巧：
**鼠标在头像里**：隐藏任务栏和按钮

**鼠标在头像外**：显现任务栏和按钮

**鼠标双击头像**：可以查看当前作业进度

**鼠标点击进度条**：返回默认状态

**鼠标点击任务栏**：显示使用说明

**鼠标按住头像不放**：可移动图标位置

### 使用问题：
**1, Word、Excel、PowerPoint等演示文件无法自动完成**：
请把云班课弹出式窗口和重定向权限设置为允许。捷径：浏览器的搜索框最左侧的拦截弹出式窗口设置为始终允许。路径：设置—隐私设置和安全性—网站设置—近期活动—www.mosoteach.cn—权限—弹出式窗口和重定向

**2,插件自动运行后发现还有作业没有自动刷完**：
以为你的网路速度太慢所导致，更换网路，刷新网页页面重新点击GO按钮或者选择其他模式点击GO按钮
